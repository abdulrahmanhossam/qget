# qget - The Minimalist CLI Video Downloader

A Go-based CLI tool that simplifies video downloading with zero manual setup.

## Key Features

- **Auto-detection and installation** of `yt-dlp` and `Deno`
- **Interactive Quality Selection** (powered by `survey`)
- **Intelligent Playlist Support** - download single videos or entire playlists
- **Cross-platform support** (Linux/Windows)

## Usage

```bash
go run main.go "<video-url>"
```

## Requirements

Only Go installed - the tool handles the rest.
